# 일상의 행복한 변화 : Smart Home 🏡

## 프로젝트 목적
* PLC 기반의 자동 제어와 HMI를 통해 실시간 모니터링, 원격제어를 결합하여 ***에너지 효율을 높이고, 가정의 편의성 및 안정성 강화***
* 최적화된 회로를 설계하고자 하며, 각 팀원들의 자동화 시스템 설계 및 프로그래밍 역량을 향상시키는 것을 목표로함
